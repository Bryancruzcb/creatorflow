# Poly Haven Dutch Ship assets

The following source packages were downloaded from Poly Haven and packaged as self-contained GLBs without changing their geometry or textures:

- Dutch Ship Large 01: https://polyhaven.com/a/dutch_ship_large_01
- Dutch Ship Large 02: https://polyhaven.com/a/dutch_ship_large_02

Poly Haven publishes all asset files under CC0 1.0. The authoritative license page is:

https://polyhaven.com/license

Authors credited by the Poly Haven asset records:

- James Ray Cock — model, textures, cleanup
- Nicolò Zubbini — original model
- Rico Cilliers — sails model and textures

The 4K glTF packages were downloaded through Poly Haven's public asset API and packaged into GLB containers using glTF Transform. The source texture and geometry payloads are preserved.
